# Bachelor Capstone Projects

This repository serves as the central hub for my final-year engineering work at Yıldız Technical University. It captures two major projects I developed, focusing on offensive security, C++ internals, and automated vulnerability assessment.

## Projects

### 1. Final Project Work: Advanced Code Obfuscation
**Focus:** Mathematical Engineering & Offensive Security
**Location:** `/final-project-work`

This project is my graduation thesis. It investigates the mathematical logic behind machine-code mutation to bypass static analysis. 
* **Key Work:** I developed C++ tools to implement matrix branching programs and XOR-equivalent transformations.
* **Content:** Baseline and mutated implementations (organized in `/blue` and `/red`), alongside my final thesis documentation and presentation.

### 2. Multidisciplinary Design Project
**Focus:** AppSec & Automated Exploitation
**Location:** `/multidisciplinary-design-project`

For this project, I built a security testing environment and custom tools to automate vulnerability research.
* **Key Work:** Custom NoSQL injection scanner and dumper scripts, paired with a vulnerable-by-design web application for testing payloads.
* **Data:** A comprehensive dataset of pentest reports, host records, and vulnerability logs in CSV and JSON formats.
