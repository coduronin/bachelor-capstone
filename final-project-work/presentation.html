Problem Statement

Modern applications are vulnerable to the Man-At-The-End (MATE) threat model, where the adversary possesses full access to the execution environment.

Traditional perimeter defenses like firewalls fail here. Static decompilation tools allow attackers to trivially reconstruct proprietary algorithms.

We aim to transition from "perfect security" to Architectural Resilience, forcing the attacker into a non-terminating analysis loop.

Framework Architecture

A cumulative pipeline where each phase reinforces the previous layer to maximize complexity.

🔣 Phase 1: Lexical Scrambling
🔄 Phase 2: Control Flow Flattening
🔐 Phase 3: Opaque Predicates
🧬 Phase 4: Metamorphic Rewriting

Phase 1 & 2: Structural Dismantling

Lexical Scrambling: Destroys semantics by mapping identifiers to cryptographic hashes.

Control Flow Flattening: Dismantles the hierarchy. The execution path is flattened into a single-level dispatcher, destroying "Visual Logic".

// Flattened Dispatcher
while (swVar != 0) {
    switch (swVar) {
        case 1: swVar = 4; break;
        case 4: swVar = 0; break;
    }
}

Phase 3: Opaque Predicates

We inject mathematically irreducible guards. Symbolic execution engines like KLEE are forced into a "Path Explosion".

They attempt to explore bogus paths mathematically guaranteed to be dead.

// Bi-Opaque Predicate
if ((7*(vx*vx)-1) != (vy*vy)) { 
    // True path
    v_target = 0x1A8C;
} else {
    // Infeasible dead code
    v_target = 0xDEAD;
}

Phase 4: Metamorphic Rewriting

Binary mutates its own instructions at runtime.

We resolve EIP via CALL/POP, then perform in-place XOR decryption on the machine code.

asm volatile (
    "call get_eip\n\t"
    "pop %%edx\n\t"
    "add $15, %%edx\n\t"
    "xorb $0x28, (%%edx)\n\t"
    ::: "memory", "edx"
);

Demonstration: Before vs After

The original logic is transparent and easily parsed by any tool.

The obfuscated version hides the state, masks the operation, and forces the analyzer to resolve the machine manually.

// Original:
if(x > 5) return 1;
return 0;
            
// Obfuscated Dispatcher:
switch(state) {
    case 1: s = (x > 5) ? 2 : 3; break;
    case 2: return 1;
    case 3: return 0;
    default: state = 0;
}
